Swiss Pharma c'est quoi ?


Swiss Pharma est un projet scolaire de groupe. Cas créé par nos professeurs, nous devions travailler pour un client de l'industrie pharmaceutique nommé Swiss Pharma. Notre mission consistait à créer un site web pour permettre la gestion des fiches de frais des agents par le service comptable. En combinant nos compétences en développement web et en design d'interface utilisateur, nous avons développé une solution web nommée "applis frais".



Les technologies qu'on a pu utiliser au cours de ce projet sont :


  • Langage de programmation : PHP, JavaScript
  • Base de données : MySQL, SQL
  • Framework : font awesome, bootstrap

Comment cela fonctionne ?


  • Page de connexion


Nous avons tout d'abord la page de connexion du projet Swiss Pharma qui permet aux utilisateurs de s'identifier et d'avoir accès à différentes pages selon qu'ils soient simple utilisateur ou comptable.


  • Page de détail


La page de détail d'une fiche de frais offre une vue de consultation des informations aux frais. Elle affiche les frais forfaitaires et hors forfait, elle propose un formulaire de modification permettant aux utilisateurs de mettre à jour les détails des frais.


  • Page d'ajout de frais


La page d'ajout donne accès une interface simple avec un formulaire pour saisir les frais forfaitaires et hors forfait d'une nouvelle fiche de frais.


  • Page de profil


La page de profil permet d'accéder à une interface personnalisé aux utilisateurs pour gérer leurs informations personnelles. Depuis cette page, les utilisateurs mettent à jour leurs coordonnées, leurs identifiants, leurs mots de passe ou d'autres paramètres.



  • Page d'aide


La page d'aide permet de répondre aux questions récurrentes des utilisateurs. Les questions se présente sous forme d'accordéon que l'utilisateur pourra dérouler pour avoir réponses à ces questions
