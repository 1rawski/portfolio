Programme d'automatisation Pyton



J'ai éffectué ce programme d'automatisation lors de mon deuxième stage chez BKS Formation. La mission qui m'était donnée consistait à créer un programme à l'aide du langage Python, qui pourra permettre au personnel de BKS Formation, d'inscrire automatiquement les nouveaux élèves de leur formation VTC, en donnant le seulement le nom du nouvel élève au programme python.



Les technologies que j'ai pu utiliser au cours de ce projet sont :


  • Langage de programmation : Python
  • Bibliothèques : pyautogui, pandas, time
  • SoftWare : Visual Studio

Comment cela fonctionne ?


  • Création du compte de l'élève


La première étape est de renseigner les données de l'élève pour pour pouvoir l'inscrire à l'examen VTC, le programme va donc dans un premier temps se connecter au compte de l'entreprise.




il va ensuite récupérer les données de l'élève qui sont stocké dans un fichier excel ou l'on rentre manuellement les données comme la ville, le code postal, la date de naissance etc...




Une fois les données pris en compte par le programme python, ce dernier va les saisir dans les champs requis.



  • Glisser les dossiers


Après avoir saisis les données de l'élève, le site demande d'importer des documents comme le permis, la carte d'identité, etc... Le programme se charge donc de plusieurs étapes :

  • Cliquer sur la fenêtre du premier document
  • Copié collé le chemin d'accès ou se trouve les documents
  • Cliquer sur le fichier demandé et validé

Ces étapes seront ensuite répétées pour les autres documents à fournir.


Le résultat

Après plusieurs semaines passé sur ce programme, je me suis rendu compte que la mission de mon tuteur était tout simplement impossible car, lors de l'étape ou le programme doit collé le chemin d'accès dans le navigateur de l'ordinateur, juste après que le programme confirme le chemin d'accès ce dernier se ferme et affiche un message d'erreur.




Apprentissages

Malgré l'échec de cette mission dû à l'infaisabilité du projet, j'ai pu approfondir mes connaissances en langage Python et utiliser de nouvelles Bibliothèques. Au delà du code, j'ai pu être autonome tout au long du stage, je devais rendre des comptes-rendus à mon tuteur à chaque fin de semaines pour que l'on voit ensemble l'évolution du projet d'automatisation.
