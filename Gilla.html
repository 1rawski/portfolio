Gilla c'est quoi ?


Le projet Gilla est un projet scolaire visant à améliorer la gestion des incidents au sein du lycée Louis Armand. Réalisé à l'aide du framework PHP Joomla et d'une architecture MVC, il repose sur une base de données MySQL. Cette plateforme, créée pour les membres du personnel et aux étudiants, offre un moyen efficace de signaler et de traiter les incidents. Les utilisateurs pourront créer des tickets d'incidents que les agents pourront ensuite prendre en charge.

Fonctionnalités


La page de connexion du projet Gilla permet aux utilisateurs de s'identifier et d'avoir accès à différentes pages selon qu'ils soient membres de l'école, agent, responsable ou administrateur.


La page d'accueil présente deux menus de navigation. Le premier menu propose la page de contact, la page d'informations du lycée, la page d'informations de l'utilisateur et la page des événements du lycée. Quant au deuxième menu, il dirige directement les utilisateurs vers la page des incidents, où ils peuvent soumettre, consulter et gérer les incidents en toute simplicité.


La page de liste des incidents du projet Gilla offre une vue d'ensemble détaillée de tous les incidents ouverts au lycée Louis Armand. Cette page permet aux utilisateurs de parcourir les incidents enregistrés, de filtrer les résultats selon différents critères ou de rechercher un incident spécifique.


Les pages détails et la page "Signaler un Incident" permettent aux utilisateurs de soumettre ou de voir les détails sur les incidents du lycée via des formulaires simples et intuitifs.



L'interface d'administration du projet Gilla propose un ensemble de pages indispensables pour la gestion des incidents. Parmi celles-ci figurent les pages "Agents", "États", "Affectations", "Emplacements", et d'autres. Ces pages permettent aux administrateurs de gérer les utilisateurs, les statuts des incidents, les affectations des tâches, les emplacements concernés, etc...



L'application mobile permet d'accéder l'essentiel, elle propose notamment la page des incidents et la page des événements. Les utilisateurs peuvent donc signaler depuis leurs smartphones, des incidents en temps réel.

Cette version Android a été développée en Java, accompagnée d'une API Joomla permettant un accès depuis les smartphones aux fonctionnalités essentielles de l'application Gilla.

Les tests

Nous avons éffectué des tests à renseigner dans des fiches. Chaque fiche est conçue pour tester une fonctionnalité, nous devons ensuite signaler si le résultat n'est pas celui attendu. Grâce à ces fiches, nous avons pu identifier les problèmes dans le front-end et back-end.




Apprentissages

Grâce à cette experience éffectué sur l'année scolaire, j'ai pu utiliser des technologies comme le framework PHP Joomla, le langage de programmation Java pour Android, et MySQL pour la base de données. J'ai pu aussi développé des compétences en modélisation de base de données et en planification d'interface utilisateur. Ce projet m'a également permis de découvrir l'architectures MVC et les API, tout en apprenant à travailler en équipe sur un projet complexe grâce à l'outil GitHub.
